# 🏨 SmartStay - Scalable Online Hotel Booking & Reservation System

A Spring Boot REST API based backend application for an Online Hotel Booking System.  
This application allows Admins, Customers, and Hotels to manage bookings, rooms, and reservations efficiently.

---

# 🚀 Tech Stack

- Java (Core Java)
- Spring Boot
- Spring Data JPA
- Hibernate
- Spring Validation
- MySQL
- Lombok
- JUnit
- Swagger UI


# 📦 Modules

- Login & Logout Module
- Admin Module
- Customer Module
- Hotel Module
- Room Module
- Reservation Module


# 🔐 Admin Features

- Add hotel / room
- Delete hotel / room
- View all hotels and rooms


# 👤 Customer Features

- Register
- Login
- Delete account
- Search hotels / rooms
- View hotel & room details
- Make reservation


 # 🏨 Hotel Features
 - Register
- Login
- Manage hotel amenities
- Add room types and descriptions
- Manage hotel location details
- View ratings and reviews
- Offer promotions and discounts
